# ユビキタス言語定義（Glossary）

## ドメイン用語

| 日本語 | 英語（コード上） | 定義 |
|--------|-----------------|------|
| シフト | shift | メンバーの月ごとの出勤日程 |
| 出勤日 | work_date | メンバーが出勤する日（日付の数値） |
| 提出 | submit | メンバーがシフトを確定して送信すること |
| 未提出 | unsubmitted | シフトをまだ送信していない状態 |
| 締切 | deadline | シフト提出の期限日時 |
| リマインド | reminder | 締切1時間前に未提出メンバーへ送る通知 |

## ロール

| 日本語 | 英語（コード上） | 定義 |
|--------|-----------------|------|
| 管理者 | admin | 全員のシフト確認・締切設定が可能なユーザー |
| メンバー | member | 自分のシフトのみ入力・確認できるユーザー |

## UI用語

| 日本語 | 英語 | 定義 |
|--------|------|------|
| シフトカレンダー | ShiftCalendar | 出勤日を選択する月カレンダーUI |
| シフト一覧表 | AdminTable | 管理者が全員分を確認する表形式UI |
| 未提出者一覧 | UnsubmittedList | まだ提出していないメンバーの一覧 |

## 技術用語

| 用語 | 定義 |
|------|------|
| Web Push | ブラウザのプッシュ通知機能。Service Workerを介して届く |
| VAPID | Web Push認証に使う公開鍵・秘密鍵のペア |
| Service Worker | バックグラウンドでPush通知を受信するブラウザ上のスクリプト |
| セッション | ログイン状態を維持するサーバー側の仕組み |
